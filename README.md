# Pose-Hackaton
## Authors Abe, Jesper, Andor, Justin

## Introduction
We are following a bootcamp in Humoid Robotics. We got the task to make a RL-policy, so that the robot could follow a given pose. We made a policy that follows a pose given by isaaclab. This repository is external because we trained on our own environments instead of the environment of `The Construct`

## Results
We got 4th place of the competition. We had a big sim-to-real gap in our policy. However, we had the best sim-to-sim score! Even better than the instructors. We won a t-shirt in this competition. We learned how to deploy our code to the robot and to the sim-to-sim, which holds a lot of value for future developments.
